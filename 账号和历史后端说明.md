# 账号和历史后端说明

已新增 `server.py`，用 Python 标准库提供本地后端服务：

- 账号注册 / 登录：`/api/register`、`/api/login`
- 当前账号：`/api/me`
- 历史保存 / 读取 / 删除：`/api/history`
- 数据库文件：`org_chart.sqlite3`

启动方式：

```powershell
python server.py
```

然后打开：

```text
http://127.0.0.1:8765
```

图片识别导入使用浏览器端 Tesseract.js，首次识别需要联网加载 OCR 引擎和语言包。
